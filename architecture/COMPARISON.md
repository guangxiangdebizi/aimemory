## 三、技术路线对比

### 路线 A：纯外部记忆（MemoryBank / FadeMem / SYNAPSE / MemGPT / SCM）

```
优点：
+ 不需要改模型
+ 即插即用
+ 可更新、可删除、可解释
+ 多用户隔离简单

缺点：
- 本质是"提醒"不是"记住"
- 检索质量是瓶颈
- 上下文拼接有 token 成本
- 模型可能忽略注入的记忆
```

### 路线 B：模型内生记忆（MEMORYLLM / M+）

```
优点：
+ 记忆真正在模型参数/latent space 里
+ 不需要检索步骤
+ 更接近"模型自己记住了"

缺点：
- 需要专门训练
- 容量有限（M+ 扩展到 160k 但仍有上限）
- 写入是"糊"的，不精确
- 多用户隔离困难
- 持续更新可能导致漂移
```

### 路线 C：注意力层集成（EM-LLM / EpMAN）

```
优点：
+ 比纯文本拼接更深层的融合
+ 认知科学启发
+ 不需要改模型权重（EM-LLM）

缺点：
- 实现复杂度高
- 和具体模型架构耦合
- 扩展性待验证
```

### 路线 D：混合路线（M+ = 内生 + 外部检索）

```
优点：
+ 兼顾内生和外部的优势
+ 核心记忆内生化，长尾记忆外部化
+ 目前效果最好（160k+）

缺点：
- 系统复杂度最高
- 需要训练
- 两套系统的协调是难点
```

## 四、我们的定位

基于以上对比，本项目的技术路线建议：

**短期（3-6个月）：路线 A 增强版**
- 基于外部记忆，但加入认知科学启发的机制
- 重点做：选择性编码 + 遗忘机制 + 动态检索
- 参考：FadeMem + SYNAPSE + EM-LLM 的组合

**中期（6-12个月）：路线 C 探索**
- 在注意力层面集成记忆
- 参考：EpMAN + EM-LLM

**长期（12个月+）：路线 D 混合**
- 核心记忆内生化（MEMORYLLM/M+ 方向）
- 长尾记忆外部化（增强版 RAG）
- 认知机制贯穿全栈

## 五、未被现有论文覆盖的空白

这些是潜在的研究机会：

| 空白领域 | 说明 | 潜在价值 |
|---------|------|---------|
| **选择性编码** | 自动判断什么值得记 | 高——减少噪音记忆 |
| **记忆巩固过程** | 短期→长期的转化机制 | 高——核心认知过程 |
| **检索竞争** | 多条记忆竞争胜出 | 中——提升检索精度 |
| **动机性遗忘** | 主动压抑不需要的记忆 | 中——隐私和安全 |
| **情感标记** | 情感强度影响记忆强度 | 中——陪伴场景 |
| **跨模态记忆** | 图像/音频记忆的统一管理 | 高——多模态趋势 |
| **多用户记忆隔离** | 一个模型服务多用户 | 高——工程必需 |
